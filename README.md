































# Nginx Web Server Setup on Ubuntu

## 📌 Objective
To install and configure an Nginx web server on Ubuntu.

## 🛠️ Steps Performed

### 1. Update System
sudo apt update

### 2. Install Nginx
sudo apt install nginx

### 3. Start Nginx
sudo systemctl start nginx

### 4. Enable Nginx
sudo systemctl enable nginx

### 5. Verify Status
sudo systemctl status nginx

### 6. Access in Browser
http://localhost

## 🌐 Output
Nginx welcome page displayed successfully.

## 📚 Learnings
- Linux command line usage  
- Package management (apt)  
- Web server setup  
- Basic deployment workflow
